📊 Google Sheets Auto-Sync Setup
1
Open Google Sheets and create a new blank spreadsheet. Save the URL of your sheet for later access.
2
Go to Extensions → Apps Script. Delete all existing code in the editor, then paste the full script below.

      
3
Click Save (💾), then Deploy → New deployment. Set Type: Web app · Execute as: Me · Who has access: Anyone. Click Deploy and copy the Web App URL.

      
4
⚠️ Important — initialise your sheet: Open the copied Web App URL directly in your browser. You should see {"status":"ok","message":"JobTrack sheet is ready!"}. This creates the headers automatically.

      
5
Paste both URLs in the fields below and click Save & Connect. Every application you add will sync instantly! ✅
